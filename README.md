# vue-beast-form

## 介绍

vue3用json方式布局并提供表单数据处理，搭配现有市场上的框架

下列是文档提供了配置的框架，诺不在配置中可自行参考后配置

- ant-design-vue
- element-plus
- vant


## 安装教程

npm
```
npm install vue-beast-form
```

pnpm
```
pnpm add vue-beast-form
```


## 文档 

github文档地址：[这里](https://menkeydyvh.github.io/vue-beast-form/)
